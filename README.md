# To-Do List App

A simple To-Do List built with HTML, CSS, and JavaScript.  
Add tasks, mark them as completed, and delete them. Tasks are saved in local storage so they stay even after reloading the page.

## Features
- Add, complete, and delete tasks
- Smooth animations for adding and removing tasks
- Responsive design

## How to Use
1. Open `index.html` in your browser.
2. Type a task and click "Add".
3. Click a task to mark it completed.
